# App Store Screenshot Generator - Examples

This directory contains example configurations and screenshots to help you get started.

## Example Configuration Files

### Basic Configuration
See `screenshots_config_basic.json` for a simple configuration with minimal options.

### Advanced Configuration
See `screenshots_config_advanced.json` for a full-featured configuration with all options.

## Example Screenshots

Place your example screenshots in this directory to test the generator.

## Usage

1. Copy an example configuration file
2. Modify paths to point to your screenshots
3. Customize titles, taglines, and colors
4. Run the generator:
   ```bash
   python ../scripts/generate_screenshot.py --config screenshots_config_basic.json
   ```
